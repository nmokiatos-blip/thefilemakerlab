# The FileMaker Lab

Production-ready Next.js site for The FileMaker Lab.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Static content for v1
- Vercel-ready

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

Editable placeholder content lives in `data/site.ts`. Shared UI components live in `components/`.
